Metadata

Metadata are an essential part of an ImGoaTeX presentation, they define elements such as title, subtitle, and author of the presentation, which are not optionnal.

They also provide the ability to set optionnal options such as date, aspect ratio, colors and default font size.


# Obligatory Metadatas

%title: My presentation
%subtitle: Optional subtitle
%author: Your Name

# Optionnal Metadatas

%date: 29-08-2004
%as_w: 4
%as_h: 3
%bgcolor: #FAFBFFFF
%color1: #1D3DB1FF
%color2: #152C80FF
%color3: #6983DFFF
%color4: #070E29FF
%basefontsize: 1.5

Sections and Subsections

Sections and Subsections are necessary to a presentation. Each frame needs to be inside a subsection, and each subsection needs to be inside a section


\section{Section title}
\subsection{Subsection title}

Frames

Frames are the heart of the presentation, they are the slides. They need to have a title, and can optionnaly have a subtitle. Frames can also have options and animations (explained after)


\begin{frame}{Frame title}{Optionnal Frame Subtitle}
    ...
    ...
    ...
\end{frame}

Frame Options

Frame can optionnaly be given the options : align = [center, top, bottom, left, right, top-left, left-top, top-right, right-top...]

Frame can also optionnaly be given animations ("In" animations and "Out") : MoveLeft, MoveRight, MoveTop, MoveBottom, Zoom, Rotate, None


\begin{frame}{Title}[align=center]<ZoomIn, MoveTopOut>
    ...
    # Animated frame with the content centered
    ...
\end{frame}

Equations

Equations are LaTeX's code inside $$ or $$...$$ like in regular LaTeX


$\int_{\mathbb{R}} e^{-x^2} dx = \sqrt{\pi}$

Pauses

Pauses allows to have multiple states of a frame separated by pauses. Thesehave by default the animation : None (<NoneIn, NoneOut>), but can be given animations the same way frame get theirs


\begin{frame}{Title}[align=center]
    ...
    ...
    \pause
    ...
    ...
    \pause<MoveRightIn>
    ...
    ...
\end{frame}

Images

Images are a piece of media, they can be gif, png, jpg... but they need to be placed inside the "medias" folder. Images can have non-necessary options : position, size, rotate, shift


\image{figure.png}[position=top-left, size==, shift=10+5+12+4, rotate=-10]

Videos

Videos are a piece of media, they thus also need to be in "medias", and can have the same options as images


\video{demo.mp4}[position=right-bottom, size=3]

Textbox

Textbox can have the same options as images and video, but can also have the option : fontsize


\textbox{Hi ! I am coding with ImGoaTeX}[position=left, shift=10+0+0+0, fontsize=2]

Iframe

Iframe can have the same options as images and video, but can also have the option, but they also need a link

WARNING : some website do not support iframe, and having an iframe will need you to have an internet connexion in order to have the website showing


\iframe{https://docs.google.com/spreadsheets/d/e/2PACX-1vSZTHvJ0v4zfKEgXnaJE9uOrBxA_G3s4HqdSXxLnRaVgUaI4zpR5t4--YcFkCfNkmGTHR6fAtay6tEW/pubhtml}[size=2, position=left-bottom]

CodeBlocks

CodeBlocks can have the same options as images and video, but can also have the option, as well as the language option (will be auto-detected otherwise). The source code needs to be in the "medias" folder"


\codeblock{main.py}[language=python, size=1]

CodeLine

CodeLines displays a single line of code, the language is auto-detected by default, but can be forced by using the option language


\codeline{ MyList = [i for i in range(0,10)] }[language=python]
