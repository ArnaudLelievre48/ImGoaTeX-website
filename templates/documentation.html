Metadata

Metadata are an essential part of an ImGoaTeX presentation, they define elements such as title, subtitle, and author of the presentation, which are not optional.

They also provide the ability to set optional options such as date, aspect ratio, colors and default font size.


# Obligatory Metadata

%title: My presentation
%subtitle: Optional subtitle
%author: Your Name

# Optional Metadata

%date: 29-08-2004 # auto-generated if not specified
%as_w: 4
%as_h: 3
%bgcolor: #FAFBFFFF
%color1: #1D3DB1FF
%color2: #152C80FF
%color3: #6983DFFF
%color4: #070E29FF
%basefontsize: 1.5
%font: Arial

Sections and Subsections

Sections and Subsections are necessary to a presentation. Each frame needs to be inside a subsection, and each subsection needs to be inside a section


\section{Section title}
\subsection{Subsection title}

Frames

Frames are the heart of the presentation, they are the slides. They need to have a title, and can optionaly have a subtitle. Frames can also have options and animations (explained after)


\begin{frame}{Frame title}{Optionnal Frame Subtitle}
    ...
    ...
    ...
\end{frame}

Frame Options

Frame can optionaly be given the options : align = [center, top, bottom, left, right, top-left, left-top, top-right, right-top...]

Frame can also optionaly be given animations ("In" animations and "Out") : MoveLeft, MoveRight, MoveTop, MoveBottom, Zoom, Rotate, None

List of supported Frame options
  • align=center
  • align=top
  • align=bottom
  • align=left
  • align=right
  • align=top-left # or left-top
  • align=top-right# or right-top
  • align=bottom-left# or left-bottom
  • align=bottom-right# or right-bottom
List of supported Frame animations

An animation can be for both "In" or "Out"

The default animation is "Fade"

  • Zoom : <ZoomIn> # <ZoomOut>
  • Rotate : <RotateIn> # <RotateOut>
  • Fade : <FadeIn> # <FadeOut>
  • None : <NoneIn> # <NoneOut>
  • MoveTop : <MoveTopIn> # <MoveTopOut>
  • MoveBottom : <MoveBottomIn> # <MoveBottomOut>
  • MoveLeft : <MoveLeftIn> # <MoveLeftOut>
  • MoveRight : <MoveRightIn> # <MoveRightOut>

\begin{frame}{Title}[align=center]<ZoomIn, MoveTopOut>
    ...
    # Animated frame with the content centered
    ...
\end{frame}

Equations

Equations are LaTeX's code inside $$ or $$...$$ like in regular LaTeX


$\int_{\mathbb{R}} e^{-x^2} dx = \sqrt{\pi}$

Pauses

Pauses allows to have multiple states of a frame separated by pauses. Thesehave by default the animation : None (<NoneIn, NoneOut>), but can be given animations the same way frame get theirs

List of supported Pause animations (same as Frame animations)

An animation can be for both "In" or "Out"

The default animation is "Fade"

  • Zoom : <ZoomIn> # <ZoomOut>
  • Rotate : <RotateIn> # <RotateOut>
  • Fade : <FadeIn> # <FadeOut>
  • None : <NoneIn> # <NoneOut>
  • MoveTop : <MoveTopIn> # <MoveTopOut>
  • MoveBottom : <MoveBottomIn> # <MoveBottomOut>
  • MoveLeft : <MoveLeftIn> # <MoveLeftOut>
  • MoveRight : <MoveRightIn> # <MoveRightOut>

\begin{frame}{Title}[align=center]
    ...
    ...
    \pause
    ...
    ...
    \pause<MoveRightIn>
    ...
    ...
\end{frame}

Images

Images are a piece of media, they can be gif, png, jpg... but they need to be placed inside the "medias" folder. Images can have non-necessary options : position, size, rotate, shift

List of supported Image options

position

  • position=center
  • position=top
  • position=bottom
  • position=left
  • position=right
  • position=top-left # or left-top
  • position=top-right# or right-top
  • position=bottom-left# or left-bottom
  • position=bottom-right# or right-bottom

shift

values are given in unit_x and unit_y, where 100 unit_x is the whole width of a slide (the same goes for unit_y)

  • shift = {top}+{right}+{bottom}+{left}

rotate

  • rotate={degres}

size

size controle the width of the media

  • size=1 # big
  • size=2
  • size=3
  • size=4 # very small

\image{figure.png}[position=top-left, size==, shift=10+5+12+4, rotate=-10]

CodeBlocks

CodeBlocks can have the same options as images and video, but can also have the option, as well as the language option (will be auto-detected otherwise). The source code needs to be in the "medias" folder"

List of supported CodeBlock options

position

  • position=center
  • position=top
  • position=bottom
  • position=left
  • position=right
  • position=top-left # or left-top
  • position=top-right# or right-top
  • position=bottom-left# or left-bottom
  • position=bottom-right# or right-bottom

shift

values are given in unit_x and unit_y, where 100 unit_x is the whole width of a slide (the same goes for unit_y)

  • shift = {top}+{right}+{bottom}+{left}

rotate

  • rotate={degres}

size

size controle the width of the media

  • size=1 # big
  • size=2
  • size=3
  • size=4 # very small

language

no language defined => the language is being guess

  • language = {language}

\codeblock{main.py}[language=python, size=1]

CodeLine

CodeLines displays a single line of code, the language is auto-detected by default, but can be forced by using the option language

List of supported CodeLine options

language

no language defined => the language is being guess

  • language = {language}

\codeline{ MyList = [i for i in range(0,10)] }[language=python]
